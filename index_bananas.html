BANANAS BSVK TESTER

Replay round-2 BANANAS books, benchmark a candidate strategy against four competitive bots tuned to the BANANAS analysis, and expose the latest executed BANANAS trade in real time from the simulator itself.
Round 2 Days -1 / 0 / 1Realtime Sim Trades4 Competitive Bots
How it works: upload BANANAS price files and a Python strategy containing strategy(row, position, step). The tester uses a hidden internal fair-value model based on the BANANAS book, and exposes the latest executed BANANAS trade as last_trade_price and last_trade_qty from actual simulated executions. PnL is scored as Cash + Position * Hidden Fair.

This tester expects the round-2 day -1, 0, and 1 price files. Testers do not upload trade CSVs. The last_trade_* fields are generated live from simulated executions.

Strategy Contract

# Your .py file must define: def strategy(row, position, step): # row -- BANANAS order book snapshot # position -- your current BANANAS inventory # step -- integer timestep index best_bid = row["bid_price_1"] best_ask = row["ask_price_1"] return { "bid": best_bid, "ask": best_ask, "bid_qty": 2, "ask_qty": 2, }

Trade source
last_trade_* updates only when the simulator itself executes a trade in the shared book. There is no uploaded or preloaded trade tape.
